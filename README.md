🚀 Built a Secure Real-Time Sales Team Dashboard with React & Supabase

🔗 Live Demo:
https://sales-team-dashboard-lilac.vercel.app/

Tech Stack:
• React JS
• Supabase (Auth + Database + Realtime)
• Row Level Security (RLS)
• React Charts

Authentication & Security:
• Sign Up / Sign In system
• Forgot & Update Password functionality
• Only authenticated users can access and control data
• Secure database operations using RLS policies

Core Features:
• Add, update, and delete sales team members
• Real-time data updates across the dashboard
• Live charts that reflect changes instantly
• Top 4 sellers dynamically displayed in charts for focused insights

User Management System:
• Dedicated Users Page
• Search users functionality
• Pagination for better performance and scalability
• All updates synced in real-time

What I Focused On:
Building strong application logic
Authentication flows in modern apps
Secure data handling with RLS
Managing real-time state in React
Scalable features like search, pagination & filtered insights
